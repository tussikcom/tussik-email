# Templating Format

#### Overview

#### Template: Page

```javascript
template = {
    'page': {
        "width": 800,
        "border": {},
        "font": {}
    }
}
```

#### Template: Footer

```javascript
template = {
    'footer': {}
}
```

#### Template: Sections

```javascript
template = {
    'sections': []
}
```

#### Template: Segments

```javascript
template = {
    'sections': [
        {
            "segments": [
                {
                    "width": "50%"
                }
            ]
        }
    ]
}
```
